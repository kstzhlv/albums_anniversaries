# Recallbum

Recallbum это небольшой CLI-инструмент на Python, который превращает профиль пользователя на [Album of the Year](https://www.albumoftheyear.org/) в напоминания о юбилеях любимых альбомов.

Проект парсит самые высоко оценённые релизы выбранного пользователя, получает дату релиза для каждого альбома, сохраняет результат локально и затем сообщает, у каких альбомов годовщина сегодня, в этом месяце или известен только год выпуска.

## Зачем этот проект

Я сделал его как маленький automation-проект под личный сценарий: не держать вручную список любимых альбомов и не забывать про их юбилеи.

Для портфолио этот репозиторий показывает:

- проектирование простого CLI с подкомандами
- парсинг HTML-страниц стороннего сайта
- извлечение, нормализацию и локальное сохранение данных
- понятный pipeline: `fetch -> parse -> transform -> remind`

## Возможности

- парсинг публичного профиля пользователя Album of the Year
- фильтрация альбомов по минимальному рейтингу
- получение даты релиза с отдельных страниц альбомов
- сохранение локального кэша в `.favourite_albums.txt`
- вывод напоминаний для:
  - альбомов с годовщиной сегодня
  - альбомов, вышедших в текущем месяце
  - альбомов, для которых известен только год релиза
- сохранение даты последнего парсинга в `.env`

## Стек

- Python 3.13+
- `argparse` для CLI
- `urllib` для HTTP-запросов
- `BeautifulSoup` для парсинга HTML
- `python-dotenv` для чтения даты последнего парсинга
- `chardet` для определения кодировки ответа

## Как это работает

1. Команда `recallbum parse <username> <threshold>` открывает профиль пользователя на Album of the Year.
2. Скрипт считывает распределение оценок и определяет, сколько страниц с highest-rated альбомами нужно обработать.
3. Для каждого подходящего альбома он запрашивает отдельную страницу релиза и извлекает дату выхода.
4. Результат записывается в `src/.favourite_albums.txt`.
5. Команда `recallbum remind` читает этот файл и печатает напоминания относительно текущей даты.

## Установка

Проект настроен для работы через PDM.

```bash
git clone https://github.com/kstzhlv/albums_anniversaries.git
cd albums_anniversaries
pdm install
```

## Запуск

Важно: текущая реализация читает и пишет `.favourite_albums.txt` и `.env` относительно текущей директории, поэтому команды ниже нужно запускать из `src/`.

```bash
cd src
pdm run recallbum parse <aoty_username> <rating_threshold>
pdm run recallbum remind
```

Пример:

```bash
cd src
pdm run recallbum parse codeling 60
pdm run recallbum remind
```

## Пример вывода

```text

```

## Ограничения

- Парсер зависит от текущей HTML-структуры Album of the Year и может сломаться, если сайт изменится.
- Напоминания пока выводятся только в консоль, без desktop-уведомлений.
- Данные сохраняются в простые локальные файлы, без базы данных.
- Запросы выполняются последовательно, поэтому большой список альбомов может обрабатываться небыстро.
